Welcome to the seismologos.onine server

This server is used to host JSON data for seismologos.gr

This server is built using Express and uses a MongoDB database. It is hosted by Render.

Available Endpoints and Routes

  • Reports

    • GET /reports - Retrieve a list of all reports, paginated.
    • GET /reports/:id - Retrieve a specific report by its ID.
    • POST /reports - Create a new report.
    • PATCH /reports/:id - Update an existing report by its ID.
    • DELETE /reports/:id - Delete a report by its ID.

Feel free to explore the API endpoints listed above. Postman recommended.
